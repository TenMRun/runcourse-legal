# runcourse-legal

런코스(RunCourse) 앱스토어 제출용 공개 법적 문서 호스팅. GitHub Pages로 배포되는
정적 페이지만 담는다 — 빌드 도구나 백엔드 연동 없음, admin-web과 완전히 분리.

- `index.html` — 개인정보처리방침 (Play Console "앱 콘텐츠 > 개인정보처리방침" URL)
- `checklist.html` — Play Console 앱 콘텐츠 9개 항목(로그인 세부정보/광고/콘텐츠
  등급/타겟층/데이터 보안/정부 앱/금융 기능/건강) 답변 근거 정리. 콘솔 자체 폼이라
  링크로 대신 제출할 수는 없고, 참고/기록용.

## 내용 갱신 시 주의

이 문서들은 `composeApp`이 실제로 수집하는 데이터(위치, 카메라, Supabase 연동 등)를
근거로 작성된 초안이다. 데이터 수집 범위가 바뀌는 앱 변경(로그인 기능 도입, 리뷰
신고 기능 추가 등)이 머지되면 함께 갱신해야 한다. 스토어 제출 전 법무 검토 필요,
"[문의 이메일 입력 필요]" 부분은 실제 연락처로 교체해야 한다.

`checklist.html`에는 현재 리뷰 기능에 신고/차단 기능이 없다는 점, 카카오맵 SDK의
위치 데이터 서버 전송 여부 미확인 등 내부 정책 검토 사항이 공개적으로 적혀 있다 —
공개 저장소이므로 민감하다고 판단되면 비공개 채널로 옮기는 것도 고려할 것.
