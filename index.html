런코스(RunCourse) 개인정보처리방침

시행일: 2026-07-09

1. 총칙

런코스(이하 “앱”)는 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 앱이 실제로 수집·이용하는 정보의 범위를 안내합니다.

2. 현재 회원가입/로그인 기능 없음

앱은 현재 이메일·비밀번호 또는 소셜 로그인(구글/카카오/애플 등) 계정 기능을 제공하지 않습니다. 따라서 이름, 이메일, 비밀번호 등 회원 식별 정보를 수집하지 않습니다. 향후 로그인 기능이 추가되면 그 시점에 본 방침을 개정하고 이용자에게 별도로 고지합니다.

3. 수집하는 정보 및 수집 목적

위치정보(GPS)러닝 경로·거리·페이스 측정, 지도 표시, 코스 이탈 감지, 도장(스탬프) 근접 판정. 화면이 꺼진 상태에서도 러닝을 계속 기록하기 위해 백그라운드 위치 접근 권한을 사용합니다.
카메라AR 크리쳐 화면 표시, 도장 인증샷 촬영. 촬영된 사진은 기기 로컬 저장소에만 보관되며 서버로 전송되지 않습니다.
리뷰 작성자명코스 리뷰 작성 시 이용자가 자유롭게 입력하는 닉네임(미입력 시 기본값 “러너”). 계정과 연결되지 않는 익명 정보입니다.
러닝 세션 요약(익명)코스 완주율 등 서비스 통계 산출을 위해 코스 식별자, 시작/종료 시각, 거리, 소요시간, 시작·종료 지점 좌표만 개인 식별 정보 없이 전송됩니다. 상세 경로·페이스·칼로리 등은 서버로 전송되지 않고 기기에만 저장됩니다.
기기 푸시 토큰발급되지만 현재 서버에 등록·저장되지 않으며 실제 알림 발송에 사용되지 않습니다.

4. 개인정보의 보유 및 이용 기간

러닝 기록(경로·거리·페이스·칼로리·인증샷 등)은 이용자 기기에만 저장되며, 앱 삭제 또는 기기 내 데이터 초기화 시 함께 삭제됩니다. 서버에 전송되는 익명 러닝 세션 요약(3항 참고)은 서비스 통계 목적으로만 보관되며 개인을 식별할 수 없습니다.

5. 개인정보의 제3자 제공 및 처리위탁

앱은 광고 SDK 및 별도 분석(애널리틱스) SDK를 사용하지 않습니다. 다음 외부 서비스를 이용해 앱 기능을 제공합니다.

Supabase익명 러닝 세션 통계, 코스·도장·크리쳐 콘텐츠 데이터의 저장·조회를 위한 백엔드 인프라(데이터베이스/API 호스팅).
카카오맵(Kakao Maps SDK)지도 렌더링. 이용자의 현재 위치 좌표가 지도 표시 목적으로 전달됩니다.
한국관광공사 TourAPI관광지·코스 공공데이터 조회. 이용자 개인정보를 전송하지 않습니다.
Firebase Cloud Messaging푸시 알림 인프라(현재 서버 미등록 상태로 실제 발송에는 사용되지 않음).

6. 이용자의 권리

현재 회원 계정이 없어 “계정 삭제” 절차가 별도로 존재하지 않습니다. 기기에 저장된 모든 러닝 기록·도장·설정은 앱 삭제로 즉시 함께 삭제됩니다. 문의사항은 아래 연락처로 문의해 주세요.

7. 개인정보의 안전성 확보조치

위치·러닝 기록 등 민감 정보는 이용자 기기 내부 저장소에만 보관됩니다. 서버로 전송되는 정보는 개인 식별자 없이 익명으로 처리되며, 서버 접근은 접근 제어가 적용된 인프라(Supabase Row Level Security)를 통해 제한됩니다.
